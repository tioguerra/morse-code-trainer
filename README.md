# Dah · Morse Code Trainer

A free and open-source mobile/tablet web app for learning and practicing
International Morse code, the playful way.

- 🔑 Pure tap/hold input — no typing keyboard, ever
- 🎓 Memrise-style spaced repetition (4h → 12h → 24h → 6d → 12d → 48d → 96d → 6mo)
- 🌳 Visual Morse decoding tree
- 🎙️ Real CW audio (sine tone with soft envelope)
- 📱 PWA — installs on any phone or tablet
- 💾 100% local-first — your progress stays on your device
- 🌍 UI in Português, English, Español, 中文
- 🆓 MIT licensed, no ads, no tracking, no paywalls

## Stack

SvelteKit · TypeScript · Tailwind v4 · Web Audio API · Dexie (IndexedDB)

## Run locally

```sh
npm install
npm run dev
```

Then open <http://localhost:5173>.
